# BeMarquee Marquee Plugin

## Features
- Infinite loop
- Auto fill
- Gap control
- Pause on hover
- Pause on click
- Reverse on hover
- Start delay
- Vertical mode

## Usage

### Vanilla
<script src="vanilla/bemarquee.js"></script>

### jQuery
<script src="jquery/bemarquee.jquery.js"></script>

## HTML Markup
```
<div class="be-marquee" data-settings='{"direction":"left"}'>
  <div class="be-marquee-inner">
    <div class="marquee-item">Item</div>
  </div>
</div>
```
## Options
direction, speed, gap, loop,
pause_on_hover, pause_on_click,
reverse_on_hover, start_delay, vertical
